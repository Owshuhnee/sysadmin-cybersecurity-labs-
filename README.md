# System Administration, Networking & Cybersecurity Labs

This repository contains hands-on System Administration, Networking, and Cybersecurity labs completed along with Software Development Course Year 2.

Each lab focuses on practical skills, clear documentation, and evidence-based learning.

## Lab Structure
Each lab includes:
- Objective and scope
- Environment details
- Step-by-step tasks
- Screenshots as evidence
- Configuration files or scripts (where applicable)
- Security considerations and reflections

## Labs
- Lab 1: Virtualisation Fundamentals
- Lab 2: Linux User & Group Management
- Lab 3: Networking Basics
- Lab 4: Services & Process Management
- Lab 5: Security Hardening

> This repository is actively updated as labs are completed.
